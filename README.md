# mbs-task – тестовая задача по [требованиям](https://maxbitsolution.com/frontend/technical_task/)

## Скрипты

- `npm run dev` — запуск локально
- `npm run build` — сборка
- `npm run preview` — предпросмотр сборки
- `npm run backend:install` — установка репозитория бэкенда (сабмодуль)
- `npm run backend:start` — запуск Cinema Backend API на `http://localhost:3022`
- `npm run test` — запуск unit-тестов (vitest)
- `npm run check-circular` — запуск проверки циклических зависимостей в проекте

## Backend (сабмодуль)

Бэкенд — подключён как сабмодуль: `cinema-backend-api`  
[maxbit-solution/frontend_technical_task](https://github.com/maxbit-solution/frontend_technical_task)

Запуск (из корня проекта):

```bash
npm run backend:install
npm run backend:start
```

## Запуск
